Description

The idea comes from one of my previous project in the world of NFTs and Web3.
This Ecommerce has the same functionalities of other online shops with the only difference that the payments are peformed using the ethereum blockchain and Metamask wallet.
The database will not be stored in the blockchain but in a SQL database which will store data about products, users, payments and pictures.
All the payments transactions will be reflected on the ethereum blockhain implemented on a test network thanks to the help of a smartcontract ( a small software stored in the blockchain ) which its funtion is to receive the payment and store it in a safe environment.
The user doesn't need an account to be able to purchase a product, he needs to connect its Metamask wallet only and in case of a purchase, provide additional information for the delivery.
Images and 'heavy' files will be stored on Amazon S3 (Simple Storage Service) whihc provides object storage through a web service interface.

Below are displayed the main actions which can be performed

Admin

  • See products list
  • Upload products
  • Modify products
  • See orders
  • See payments
  • Generate financial reports
  • Refund client
  • Upload expenses

User

  1. Connect Metamask wallet
  2. Update user profile
  3. Open product
  4. Add product to cart
  5. Pay in ethereum
  6. Wait payment confirmation
  7. Display order
  8. Open ticket
